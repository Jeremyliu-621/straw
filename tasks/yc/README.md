## Conventions

- The application draft is **load-bearing**. When the pitch shifts, update [YC_APPLICATION_DRAFT.md](YC_APPLICATION_DRAFT.md) and timestamp it in the frontmatter.
- The session transcripts are **append-only history** — don't edit. Start a v3 if a major redraft happens.
- If a number, claim, or quote in the application doesn't have evidence somewhere in [research/](../research/) or [strategy/](../strategy/), flag it. Pitches that don't survive YC's pushback questions don't survive the partner round.
- The repo is private; these drafts can stay tracked here. If the repo ever flips public, move active drafts back to gitignore (per `tasks/README.md` conventions).
