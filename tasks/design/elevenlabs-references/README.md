# ElevenLabs dashboard references

Eleven screenshots Jeremy shared as the design north star for the dashboard. ElevenLabs runs a dark theme — Straw is light — but the *patterns* (card-based pages with imagery, gradient heroes, mood-tile categories, first-visit onboarding modals, asset grids) translate either way.

For distilled patterns + how to apply them to Straw, see [../inspiration-patterns.md](../inspiration-patterns.md).
