## Conventions

When updating these docs:
1. **Date your edits.** Strategic docs decay fast; a stale TAM number is worse than no number.
2. Keep the YAML frontmatter `last_updated` field accurate.
3. If a strategic claim becomes load-bearing for a decision, link from DECISIONS.md back to the source doc here.
4. Don't duplicate research files — reference [research/](../research/) for the deep work that informed a strategic claim.
